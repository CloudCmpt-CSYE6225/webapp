# Web Application

## Prerequisites
- **MySQL Server**
- **VSCode** (or any other IDE)

## Build and Deploy Instructions

1. **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```

2. **Install dependencies**:
    ```bash
    npm i
    ```

3. **DB connection**:
    - **Create .env file and add db connection values**

4. **Run the application**:
    ```bash
    dev: npm run dev
    test: npm run test
    ```
